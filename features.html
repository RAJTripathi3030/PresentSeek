About PresentSeek

PresentSeek is a simple and easy to use attendance management system designed to improve productivity and eliminate paperwork. Remove all the tedious excel sheets from your computer and make use of our solution to keep track of student attendance. Let the system create and store excel sheets for you to use as you see fit.

The Student Page

This page Allows students to view their attendance data as soon as the attendance is marked. Also predicts how many lectures the student needs to attend to maintain the minimum criteria.

The Teacher Page

This page allows the teacher the feature of hassle-free class creation and attendance marking. Attendance records are maintained upto 30 days and can easily be downloaded upon requirement. Created Classes may be edited or removed as per the need.

The Admin Page

This page allows an institute representative to register their institute with PresentSeek. Furthermore,only the admin has the ability to create teacher and student accounts for security purposes.

Teacher Interface

Dashboard

View your details and profile picture. Make changes as needed

Create Class

Fill out the class details form in order to create a completely new class for a new term

Add Students

Add students to your created classes. Students registered with the same branch as the class can be added simultaneously.

Mark Attendance

Select an existing class and mark its students as present or absent. A .csv file is generated upon submission for record keeping

Attendance Reports

Download attendance reports by selecting the corresponding date of marking. A .csv file of the records is downloaded on your local device

View and Update Classes

View all students in your classes along with their details. Update the class details as per requirement

Remove Class

Delete a class entirely. It will no longer be available for viewing or marking. All records pertaining to said class should be backed-up on your local system

Remove Student

Remove a student from a class. The data of said student will also be removed from all records
